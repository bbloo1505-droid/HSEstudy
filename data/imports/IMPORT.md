# Import workflow

1. Download the official PDF from the Queensland Government / WorkSafe Queensland site.
2. Convert to plain text locally (no cloud OCR required if you can copy text).
3. Save as `data/documents/<slug>.md`.
4. Create `data/documents/<slug>.meta.json` with accurate status and dates.
5. Open HSE Launchpad → Official source library → **Re-import documents**.
6. Ask a question in **Source mode** and confirm the citation locator looks right.

Never upload these files to a cloud AI service. This app is local-only.
